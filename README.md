# UnitedPolitics
UnitedPolitics is a plugin for geopolitical Minecraft servers, first created for UnitedLands (play.unitedlands.net). The core feature is the tracking of reputation between geopolitical actors (nations, regions and towns) as a framework for other mechanism / plugins to hook into.

Detailed information can be found on the [Wiki](https://github.com/United-Lands/UnitedPolitics/wiki).
